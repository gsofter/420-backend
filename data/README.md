# thc_hash_table.csv

CSV file's header and data format should match with the following:


| THC | Size | Rate |
| --- | ---- | ---- |
| THC 2% |7 | -2% |

And required delimiter is `;`.

# thcHashTable.ts

This script parses the `thc_hash_table.csv` and generates the `thcHashTable.json` to be used in the system.
